# 🖥️ Caso 01 – Computador com lentidão extrema

## 📌 Descrição do Problema
Usuário relatou que o computador demorava mais de 10 minutos para iniciar, com travamentos constantes durante o uso.

## 🔍 Diagnóstico
- Verificação de uso de disco em 100%
- Identificação de HD mecânico antigo
- Sistema operacional sobrecarregado

## 🛠️ Solução Aplicada
- Backup dos dados do usuário
- Substituição do HD por SSD
- Reinstalação do Windows
- Atualização de drivers

## ✅ Resultado
- Tempo de boot reduzido para aproximadamente 20 segundos
- Sistema estável e sem travamentos

## 🧠 Aprendizados
- Impacto direto do armazenamento na performance
- Importância do diagnóstico antes da intervenção

## 🔐 Observações de Segurança
- Recomendado backup periódico para evitar perda de dados
